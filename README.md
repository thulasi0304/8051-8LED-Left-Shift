# 8051 8 LED Left Shift blinking

## Description
This project demonstrates left shift blinking of 8 LEDs using the AT89C51 microcontroller. The program is developed in Embedded C using Keil uVision and simulated in Proteus.

## Components Used
- AT89C51 Microcontroller
- 8 LEDs

## Software Used
- Keil uVision
- Proteus 8

## Files
- 8led ls.c – Embedded C source code
- 8led ls.uvproj – Keil project file
- STARTUP.A51 – Startup file
- Circuit.png – Proteus circuit diagram
